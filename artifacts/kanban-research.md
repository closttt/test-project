# Kanban board — research & improvement plan

> Ресёрч-агент упал на transient API-ошибке (response stalled), поэтому синтез сделан вручную по
> известным паттернам лидеров (Trello, Linear, TickTick, Notion board, Jira, Height, Asana,
> ClickUp, Todoist board). Где не проверено вживую — помечено «паттерн из знаний».

## Ключевые проблемы нашего канбана (сейчас)
1. **Мобильность.** Колонки фикс. 288px + горизонтальный скролл всей доски. На телефоне это узко и
   неудобно; HTML5-drag на тач ненадёжен → карточку часто **нельзя переместить пальцем вообще**.
2. **Добавление.** Карточку нельзя создать прямо в колонке — только глобальный quick-add, потом тащить.
3. **Плотность / навигация.** Нет сворачивания колонок; на доске с многими колонками всё разъезжается.

## Как делают лидеры (паттерны)
| Приём | У кого | Зачем |
|---|---|---|
| **«+ Добавить карточку» прямо в колонке** | Trello, Notion, Linear, Todoist, ClickUp | создать задачу в нужном статусе без перетаскивания — самый частый экшен |
| **Меню «Переместить в…» на карточке** | Trello (mobile), Jira mobile, ClickUp | на тач-устройствах перенос без drag — тап → выбрать колонку |
| **Свайп между колонками + одна колонка на экран** | Trello mobile, Notion mobile | мобильная навигация по доске жестом, а не узкими колонками |
| **Сворачивание колонок** | Jira, Linear, ClickUp, Height | убрать неактуальные статусы, сжать доску |
| **Ручной порядок карточек внутри колонки** | Trello, Linear, Notion, Jira | приоритезация внутри статуса, не только между |
| **Счётчик/лимит WIP в шапке** | Jira, Trello (Power-Up), Linear | контроль перегруза (у нас уже есть на «Доске») |
| **Sticky-заголовки колонок** | Jira, Linear | заголовок виден при вертикальном скролле длинной колонки |

## Приоритизация улучшений
| Улучшение | Приоритет | У кого есть | Сложность у нас |
|---|---|---|---|
| «+ карточка» в каждой колонке (создание с нужным свойством) | **must** | все | низкая |
| Меню «Переместить в…» на карточке (mobile-перенос без drag) | **must** | Trello/Jira mobile | низкая |
| Адаптивная ширина колонок + snap-свайп на мобиле | **must** | Trello/Notion mobile | низкая (CSS) |
| Сворачивание колонок | **should** | Jira/Linear/ClickUp | средняя |
| Ручной порядок карточек внутри колонки (drag) | **should** | Trello/Linear | средняя |
| Sticky-заголовок колонки при вертикальном скролле | nice | Jira/Linear | средняя |
| Свимлейны (группировка внутри колонки) | nice | Jira/ClickUp | высокая |

## Топ-5 правок «внедрить прямо сейчас»
1. **Quick-add карточки в колонке** — инпут «+ карточка» внизу каждой колонки; создаёт задачу и сразу
   присваивает свойство колонки (приоритет/проект/срок/кастом-колонка) через её `onDrop`. Кроме
   «Готово» (туда добавлять нелогично).
2. **Меню «Переместить в…»** на каждой карточке — иконка ⇄, тап → список остальных колонок → перенос.
   Делает доску реально рабочей на телефоне, где drag не срабатывает.
3. **Адаптивные колонки + snap** — на мобиле колонка ~82vw с «подглядыванием» соседней + `snap-x
   snap-mandatory` (одна колонка на экран, свайп); на десктопе прежние 288px.
4. **Сворачивание колонок** — кнопка сворачивает колонку в узкую полосу (вертикальная подпись + счётчик),
   тап разворачивает. Состояние — в localStorage (UI-преференс).
5. **Ручной порядок внутри колонки** — перетаскивание карточки на другую карточку в той же колонке
   меняет их порядок (через `order`), не только перенос между колонками.
