# Роадмап продукта — личный таск-менеджер / мини-CRM

Утверждён 2026-07-12. Источники: `brief.md`, `research.md`, `ia.md`, `competitor-teardown.md`.
Решения по развилкам: конкуренты — Todoist, Linear, Things 3, WeekToDo, Notion CRM, TickTick, Sunsama;
motion — сбалансированно (уровень Linear); режим loop — глубина (Волна 0 до идеала, затем Волна 1).

## Принципы (из research)

- **Progressive disclosure** (§1.3): дашборд = Summary-уровень, 3 крупных блока, без графиков. Детали — по drill-down.
- **Recognition over recall** (§1.3): 6 буквальных пунктов навигации, стабильный порядок.
- **Клиент как хаб** (§1.4 №3): клик по клиенту раскрывает проекты/деньги/задачи, а не плоскую запись.
- **Модель Things для подзадач** (§1.3): чек-лист внутри задачи + прогресс-индикатор Plutio + сворачивание Todoist.
- **Дашборд-дом + глобальный quick-add** (§1.3): «что видно при входе» ≠ «куда падает новая задача».
- **Порог риска 14/30 дней**, настраиваемый (§1.3).
- **Empty state = онбординг** (§1.3): одно очевидное действие, не пустая заглушка.
- **Anti-scope железно**: без графиков динамики, канбана сделок, тайм-трекинга, интеграций, мультиюзера.

## Волна 0 — «убрать сырость» (loop-фокус)

1. **Empty states** для всех разделов (Дашборд/Клиенты/Проекты/Задачи/Заметки/Календарь) — иллюстративная иконка + текст + одно CTA.
2. **Карточка-хаб клиента** — отдельный вид: контакты, деньги, статус, история активности, связанные проекты и задачи, кнопка «отметить активность».
3. **Дашборд-полировка** — money-tiles цифрами (выручка / ожидается / N клиентов в риске) + icon-chips + иерархия как скрин 1, без графика.
4. **Теги на задачах** — цветные бейджи, фильтр по тегу.
5. **Прогресс-бар чек-листа** — тонкая полоса + «3/5» на карточке задачи.
6. **Фундамент Framer Motion** (сбалансированно): fade+slide переходы страниц, stagger списков, «вкусный» чек-анимация, hover-lift карточек, spring-диалоги, active-indicator в сайдбаре.
7. **QA**: tsc чистый, dev проверен скриншотом на всех экранах, no horizontal scroll 1440/768/375.

## Волна 1 — глубина

Экран Настроек (порог риска, тема) · настоящий календарь-вид (сетка) · Command palette Cmd+K · переключатель тем light/dark · natural-language ввод дат в quick-add · фильтры/поиск в списках.

## Волна 2 — craft & delight

Drag-to-reorder задач (spring) · shared-layout анимации · count-up money-tiles · attention-pulse бейджа риска · микро-праздник при закрытии всех задач дня · сквозная клавиатурная навигация.

## Волна 3 — масштаб (позже)

Supabase бэкенд · синк устройств · авторизация. Локальный идеал раньше облака.
