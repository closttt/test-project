# CRM — version 3.3 (компактный хендофф)

Личный таск-менеджер. React + Vite + TS + Tailwind + shadcn/ui. Локал-фёрст (`localStorage`), только десктоп.

## Что нового в 3.3

Внутрянка и микро-взаимодействия — без новых разделов, только полировка существующего. Ресёрч по
конкурентам (TickTick/Todoist/Things/Notion/Linear/GitHub-heatmap) и аудит кода — в
`docs/superpowers/specs/2026-07-14-crm-v3.3-roadmap-design.md`.

- **Задачи — инлайн-редактирование**: клик по строке больше не открывает форму (только значок-карандаш /
  правый клик / клик по названию в канбане). Теги теперь редактируются прямо на строке (тот же дропдаун-
  паттерн, что у даты) — приоритет и срок уже были инлайн.
- **Проекты — приоритет + время**: `Project.priority` (клик по флажку на карточке, без формы) и авто-время
  = сумма `estimateMin` открытых задач проекта (не ручное поле).
- **Правая панель фильтров**: сортировка/теги/статус на Задачах/Проектах/Знаниях/Заметках переехали в общий
  сворачиваемый `AppShell`-риэйл (push, не оверлей). Поиск остался инлайн на месте.
- **Аналитика — hover-тултипы**: общий `useChartTooltip()` (без новых npm-пакетов), подключён к
  AreaChart/BarChart/DonutGauge/HBarList и к карте фокуса (мигрировала с нативного `title`).
- **Pomodoro — быстрые настройки**: шестерёнка в BottomDock и на плавающей панели — те же 4 поля +
  автостарт, что в Settings, без перехода со страницы.
- **База знаний — приоритет по клику**: `KnowledgeCard.priority` (новая колонка + RLS UPDATE-политика),
  `PriorityFlag` прямо на карточке/строке/в диалоге.
- **Убрано**: Инсайты (панель + `lib/insights.ts`, все точки входа) и виджет дашборда «Неделя впереди»
  (удалён из реестра, не просто скрыт).

**Диагностировано, не починено в этом проходе** (конфиг вне репозитория): фото и ссылка из Telegram не
долетают до карточек базы знаний — разрыв в кастомном скилле Hermes Agent (Hermes Desktop), не в CRM.
Схема Supabase (`image_url`/`source_url`/бакет) готова с 3.1. См. спеку §4.

Отражено в `src/lib/changelog.ts` (запись "3.3", 2026-07-14) и в `src/lib/roadmap.ts`
(`SHIPPED_TITLES`, весь пакет закрыт; пункт про Hermes-скилл остался в «Нужно решение» с уточнённым текстом).
