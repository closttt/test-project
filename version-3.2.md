# CRM — version 3.2 (компактный хендофф)

Личный таск-менеджер. React + Vite + TS + Tailwind + shadcn/ui. Локал-фёрст (`localStorage`), только десктоп.

## Что нового в 3.2

- **Дашборд — виджет «Неделя впереди»**: превью задач на ближайшие 7 дней прямо на `/`.
- **Расширенные правила повтора задач**: помимо старых интервалов — только по будням, «первый понедельник месяца».
- **Зависимости задач** (`blockedBy`): задачу нельзя закрыть, пока не закрыты все задачи, от которых она зависит (guard на завершении), в `TaskEditDialog` — иконка замка и выбор зависимостей.
- **Шаблоны проектов**: сохранение существующего проекта как шаблона (Settings), выбор шаблона при создании нового проекта (`Projects.tsx`), управление шаблонами в настройках.

Отражено в `src/lib/changelog.ts` (запись "3.2", 2026-07-13) и в `src/lib/roadmap.ts` (`SHIPPED_TITLES`, все 4 пункта закрыты).
